## 📖 About the project

Atlantic Blue Diving is a complete redesign of a fictional diving center inspired by real businesses, built entirely from scratch with its own identity, architecture and user experience.

The goal of this project is to demonstrate professional frontend development skills by combining modern UI design, accessibility, responsive development, SEO optimization and Progressive Web App technologies.

The website has been designed as if it were a real commercial project, focusing on maintainability, scalability and performance.

```bash

```

```bash

```

